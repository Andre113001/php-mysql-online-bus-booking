1: Install XAMPP
2: Copy this folder to C:\xampp\htdocs\www\
3: Import 'irosin-elavil-db' to MySQL / phpmyadmin and name the database as 'irosin-elavil-db'

User-side:
Access the website using the browser's address bar. 
Simply input: http://localhost/www/thesis-elavil/php/landing-index.php

Admin-side:
As mentioned in the user-side, use the address bar to access the admin-side
Simply input: http://localhost/www/thesis-elavil/php-admin/admin-index.php
